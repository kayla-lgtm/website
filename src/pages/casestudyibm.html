Case Study

IBM watsonx Code Assistant generated over half of Playbook content

IBM CIO Hybrid Cloud Platforms used watsonx Code Assistant for Red Hat Ansible Lightspeed to accelerate automation content creation and expand developer productivity.

Challenge

The Hybrid Cloud Platforms team within the IBM CIO organization is responsible for delivering, securing and modernizing a vast IT environment that powers business-critical applications used by hundreds of thousands of IBM employees.

Their mission extends beyond uptime. The team focuses on accelerating IBM’s broader business transformation by optimizing core infrastructure so other teams can move faster and innovate. To do that, they continuously seek ways to reduce repetitive manual operations and free developer time for higher-value modernization work.

This drove their interest in testing IBM watsonx Code Assistant for Red Hat Ansible Lightspeed during technical preview.

Solution

Before preview adoption, the team was already using Red Hat Ansible Automation extensively for patching, vulnerability remediation, and system health checks. They introduced watsonx Code Assistant for Red Hat Ansible Lightspeed into these established workflows.

With natural-language prompts, developers could generate task recommendations for Ansible Playbooks. This lowered the barrier to creating automation content and expanded potential contributors beyond the previously small group of specialists.

IBM projected that the number of people able to produce Playbook content could grow by roughly a factor of ten as enterprise adoption matured.

Results

During technical preview, approximately 60% of IBM’s initial Ansible Playbook build content was generated through watsonx recommendations.

The team described their modernization progression as crawl, walk, run, and now fly, with watsonx Code Assistant representing the next acceleration stage for automation delivery.

As with all enterprise transformations, results vary by environment, data, systems and operating conditions, but IBM’s experience demonstrated strong potential for productivity and scale.
